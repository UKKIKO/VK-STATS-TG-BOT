VK Stats Bot
Простой Telegram-бот, который показывает статистику (лайки, комментарии, репосты) последнего поста в сообществе ВКонтакте.

Как установить:
  - Скачай файлы.
  - Установи библиотеки: pip install vk_api pyTelegramBotAPI
  - Вставь свои токены в файл bot.py.

Команды бота:
  /start - Инструкция - "Введите краткое имя группы или пользователя без @ в начале".
  [любой текст] - Бот воспринимает текст как ссылку на страницу/сообщество ВК и выдает статистику - создаёт выборку из 5 последних постов и проходится по ней, пропуская все закреплённые. 
Как только сталкивается с незакреплённым - цикл по выборке оканчивается и бог выводит статистику полученного поста. Если не сталкивается - пишет сообщение о том, что не смог найти.
