VK Stats Bot
Простой Telegram-бот, который показывает статистику (лайки, комментарии, репосты) последнего поста в сообществе ВКонтакте.

Как установить:
  - Скачай файлы.
  - Установи библиотеки: pip install vk_api pyTelegramBotAPI
  - Вставь свои токены в файл bot.py.

Команды бота:
  /start - Приветствие
  /help - Инструкция
  /complex [любой текст] - Бот воспринимает текст как краткое имя страницы/сообщества ВК и выдает статистику - создаёт выборку из 10 последних постов и проходится по ней, пропуская все закреплённые. Высчитывает коэффициент вовлечённости для каждого поста по формуле (кол-во лайков/кол-во подписчиков сообщества)*100, после чего высчитывает среднее значение по всех 5 найденным постам. Также находит среди этих 5 постов "лучший" (Обладающий наивысшей суммой лайков, комментариев и репостов)
  /simple [любой текст] - Бот воспринимает текст как краткое имя страницы/сообщества ВК и выдает статистику - создаёт выборку из 10 последних постов и проходится по ней, пропуская все закреплённые. Выводит сырые данные (Просмотры, лайки, комментарии, репосты и дату) для первых 5 постов, которые находит.
